# PetID -- Product Requirements Document (PRD)

## Product

PetID -- Digital identity and health record for pets.

## Objective

Provide pet owners with a simple way to store and access their pet's
essential information and share it through a QR code.

## Key User Story

As a pet owner, I want to store my pet's health information and generate
a QR code so that if my pet is lost or I need to share information with
a veterinarian, the data is easily accessible.

## Target Users

Primary: - Pet owners - Multi‑pet households

Secondary: - Veterinarians - Pet hotels - Pet sitters

## Core MVP Features

Authentication - Email signup/login - Google OAuth - User session

Pet Management - Create pet - Edit pet - Delete pet - Upload pet photo - Mark pet as lost/found

Health Records - Vaccines - Allergies - Medical notes

QR System - Unique QR code per pet - Public pet profile

Public Pet Page - Pet photo - Breed - Allergies - Emergency contact - Lost pet banner

Report Found Pet - Finder can send message - Contact info - Optional location

Owner Settings - Edit profile (name, phone) - Notification preferences for nearby lost pets

## Non Goals (MVP)

-   Vet clinic management
-   Payments
-   Advanced analytics
-   Full mobile app

## Success Metrics

-   Number of registered users
-   Pets created
-   QR scans
-   Weekly active users
